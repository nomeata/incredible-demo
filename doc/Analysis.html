data ScopedProof Source

This function turns a proof into something that is easier to work on in unfication: * Constants have been turned into variables, * Local variables have been renamed * Scopes have been calculated * Free variables have their scoped variables as arguments

After unification, ScopedProofs contains the fully unified result
